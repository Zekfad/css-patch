    Features

    Features presented in this stub:

    • ESLint.
    • Babel.
    • Terser.
    • ESM and CommonJS input and output.
    • TypeScript declarations generation.
    • Docs generation.

    Install

    Install via yarn:

    yarn add stub

    Install via npm:

    npm i stub
